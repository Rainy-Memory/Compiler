## Mx* Language

Mx* is a simplified language from the mix of Standard C++ and Java. Details of Mx* can be found [here](https://github.com/ACMClassCourses/Compiler-Design-Implementation).

## Progress

 - [x] g4

 - [x] SymbolCollector
 - [x] ScopePrinter
 - [x] SemanticChecker
